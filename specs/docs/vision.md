# Vision: Telemetriya (умные заметки в Telegram)

## Зачем
Сделать Telegram-бота, который:
- принимает текст/голос/файлы (PDF и др.)
- сохраняет и индексирует “как память”
- умеет доставать релевантное по запросу (“что я кидал вчера про X?”)
- делает саммари, теги, классификацию
- создает напоминания/события
- интегрируется с Todoist через MCP (создание задач, проекты, дедлайны)

## ЦА
Я (личная продуктивность) + люди, кто пересылает статьи/видео/идеи и хочет потом быстро найти.

## Основные сценарии (MVP)
1) Я кидаю PDF → “Сделай саммари” → бот возвращает кратко + ключевые тезисы.
2) Я пишу “Сделай напоминание завтра 9:00 позвонить…” → бот ставит напоминание.
3) Я пишу “что я скидывал вчера про …” → бот находит и дает релевантное.
4) Я пересылаю пост/ссылку → бот предлагает: (а) в архив, (б) кратко объяснить, (в) сделать задачу в Todoist.

## Качество/метрики
- Evals pass-rate по intent/маршрутизации ≥ 90% на базовом наборе.
- Саммари: субъективно “полезно” + не теряет главные выводы (подкреплять eval кейсами).
- Поиск “вчера/на прошлой неделе” — корректные результаты.

## MVP-определение
“MVP готов”, если можно:
- отправить текст/голос → получить структурированную заметку
- отправить PDF → получить саммари
- спросить “саммари статьи, что я скинул вчера” → бот находит конкретный объект
- создать Todoist задачу из диалога
- все ключевые маршруты покрыты eval cases
